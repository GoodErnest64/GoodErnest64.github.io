اولین برنامه شما

قبل از شروع کردن برنامه نویسی باید یک چیزی را به یاد بسپارید. نرم افزار هایی که به صورت عادی میبینید از یک پنجره استفاده میکنند. اما ما فعلا خروجی نرم افزار را در console که همان cmd کامپیوتر است استفاده میکنیم اما جلوتر به ساختن پنجره نیز میرسیم.

در زبان برنامه نویسی پایتون از دستور print() برای نمایش دادن متن در کنسول استفاده میکنیم. پارامتری که قبول میکند نیز از نوع string است. اگر این اصطلاح ها را متوجه نمیشوید هیچ اشکالی ندارد جلوتر به طور کامل به آنها میپردازیم.

print("Hi. I'm Working !!!")

حال اگر این کد را اجرا کنید خروجی زیر را میبینید : 

Hi. I'm Working !!!

انواع داده :

همانطور که میدانید داده ها در کامپیوتر به صورت باینری یعنی همان0 و 1 ها ذخیره میشود. اما در برنامه نویسی برای سادگی کار ما داده ها به انواع مختلفی تقسیم شده اند.

  • int : اعداد صحیح
  • float : اعداد اعشاری
  • string : یک جمله
  • bool : درست یا غلط
  • list : یک لیست از داده های دیگر
  • dictionary : یک لیست از داده های دیگر با نام گذاری مخصوص
  • و...

در بقیه زبان های برنامه نویسی مانند c++ و java قبل از مشخص کردن متغیر نیاز است نوع آن را مشخص کنیم اما پایتون اینگونه نیست. برای مشخص کردن یک متغیر باید یک نام و مقدار آن را قرار دهیم.

a = 0 #int
b = 3.14 #float 
c = "I am a string" #string
d = True
e = [1, 2, 3] #list
f = {"item1":"apple", "item2":"orange"} #dictionary

نکته : الان شاید به خاطر # هایی که در کد گذاشتم گیج شده باشید. به آن بخش هایی که قبل آن ها هشتگ می آید کامنت میگویند و برای یادداشت گذاری استفاده میشود. نکته مهم راجع به کامنت ها این است که پایتون کامنت را نادیده میگیرد. یعنی اجرا نمیشود و فقط برای خودتان یا کسانی است که در آینده میخواهند کد را بخوانند.

محاسبات :

وقتی از کامپیوتر و برنامه نویسی حرف میزنیم قطعا بحث محاسبات نیز به میان می آید. در پایتون از عملگر های زیر برای انجام محاسبات استفاده میکنیم :

  • = : مساوی
  • + : جمع
  • - : منها
  • * : ضرب
  • / : تقسیم
  • ** : توان
  • // : ریشه
  • % : باقی مانده تقسیم
  • += : مقدار را مساوی آن عدد + عدد بعدش میکند
  • -= : مقدار را مساوی آن عدد - عدد بعدش میکند
  • *= : مقدار را مساوی آن عدد * عدد بعدش میکند
  • /= : مقدار را مساوی آن عدد / عدد بعدش میکند
  • **= : مقدار را مساوی آن عدد به توان عدد بعدش میکند
  • //= : مقدار را مساوی ریشه عدد بعدم آن عدد میکند
  • %= : مقدار را مساوی باقی مانده آن عدد تقسیم بر عدد بعدش میکند

مثال : 

a = 10
b = 20
c = 3
print(f"a + b = {a + b}")
print(f"a - b = {a - b}")
print(f"a * b = {a * b}")
print(f"a / b = {a / b}")
print(f"a ** c = {a ** c}")
print(f"a // c = {a // c}")
print(f"a % c = {a % c}")

نکته : می دانید که برای ساختن string از "" استفاده میکنیم . اما چگونه مقدار یک متغیر را print کنیم ؟ در مثال بالا من قبل از "" ها یک f گذاشتم که مخفف format است. نحوه کار فرمت این است که اگر درون متن {متغیر} ببیند به جای {} مقدار متغیر را قرار می دهد.

برای مثال : 

a = 10
print(f"a is equal to : {a}")

و خروجی این برنامه : 

a is equal to : 10

این نکته را به خوبی به یاد بسپارید چون در آینده زیاد با آن سر و کار داریم.

حال خروجی این برنامه : 

a + b = 30
a - b = -10
a * b = 200
a / b = 0.5
a ** c = 1000
a // c = 3
a % c = 1

مقایسه :

در مقایسه ها همیشه مقدار True یا False برگردانده میشود که میتواند در یک bool ذخیره شود.

  • < : سمت چپ بزرگتر از سمت راست
  • > : سمت چپ کوچکتر از سمت راست
  • =< : سمت چپ بزرگتر یا مساوی سمت راست
  • => : سمت چپ کوچکتر یا مساوی سمت راست
  • == : سمت چپ مساوی سمت راست
  • =! : سمت راست مساوی نیست با سمت راست
  • and : اگر سمت راست و چپ True بودند مقدار True بر میگرداند ام حتی اگر یکی از آنها False بود False بر میگرداند
  • or : اگر سمت چپ و راست هر دو false بودند false بر میگرداند اما اگر حتی یکی از آنها True باشد True بر میگرداند
  • not : همان !

برای مثال : 

a = 1
b = 2
print(a < b)
print(a > b)
print(a == b)
print(a <= b)
print(a >= b)
print(a != b)
c = True
d = False
print(c and d)
print(c or d)
print(not c)

شرط و شروط :

وقتی دو چیز را با هم مقایسه میکنیم میخواهیم که اگر برای مثال یکی از دیگری بیشتر بود اتقاقی بیفتد. برای این کار از if استفاده میکنیم. مثال : 

a = 10
b = 20

if a > b:
    print("A is bigger than B")

در کد بالا اگر a بیشتر از b باشد پیغام را نمایش میدهد و چون اینطور نیست پس خروجی نمایش داده نمیشود.

در مبحث شرط و شروط دو چیز مهم دیگر نیز داریم: else و elif مثال : 

a = 10
b = 20

if a > b:
    print("A is bigger than B")
elif a < b:
    print("A is smaller than B")
elif a == b:
    print("A equals B")
else:
    print("This can't Happen !")

در کد بالا اگر a بزرگتر از b باشد پیغام اول را پرینت میکند, اگر از b کوچکتر باشد پیغام دوم و اگر برابر آن باشد پیغام سوم را پرینت میکند. حالت چهارم هم هرگز اتقاف نمی افتد چون a یا بزرگتر از b است یا کوچکتر از آن است و یا برابر آن است پسپیغام چهارم هیچ وقت نمایش داده نمیشود.

خروجی : 

A is smaller than B

لیست و دیکشنری :

فرض کنید میخواهیم 1000 عدد را ذخیره کنیم. آیا حاضرید 1000 خط کد بنویسید و برای هر یک نامی بگذارید. این کار هم باعث تلف کردن زمان و هم باعث پر شدن رم و کند شدن نرم افزار میشود. برای همین از لیست ها و دیکشنری ها استفاده میکنیم. لیست ها در زبان های دیگر آرایه نیز خوانده میشوند. مثال : 

l = [1, 2, 3]

در کد بالا یک آرایه با نام l ساختیمو در آن سه مقدار 1و 2و 3 را قرار دادیم. حال شاید بپرسید چگونه به هر یک از داده های درون لیست دسترسی پیدا میکنیم؟ برای دسترسی به آنها از index استفاده میکنیم . شروع شمارش در کامپیوتر ها از 0 شروع میشود. یعنی اگر میخواهیم اولین مقدار را در لیست بگیریم باید index 0 آن را بگیریم. و یا اگر میخواهیم داده دهم را بگیریم باید از index 9 استفاده کنیم. نحوه گرفتن index نیز با [] است برای مثال : 

l = [1, 2, 3]
print(l[0])
print(l[1])
print(l[2])

خروجی : 

1
2
3

در دیکشنری ها تنها تقاوت این است که دیگر index عدد نیست و index هر داده رو خودتان مشخص میکنید. مثال: 

dic = {"health" : 20, "coins" : 7}
print(dic["health"])
print(dic["coins"])

خروجی : 

20
7

تکرار :

فرض کنید میخواهید کدی را 1000 بار تکرار کنید یا میخواهید تک تک اعداد در یک لیست را پرینت کنید. در چنین جاهایی از loop یا حلقه استفاده میکنیم. مثال : 

for i in range(10):
    print("a")

اگر به زبان های دیگر تسلط دارید این کد معادل کد زیر است (مثال ++C): 

for(int i = 0; i < 10; i++){
    cout<<"a";
}

در کد بالا ما یک for loop داریم که 10 بار تکرار میشود. در پایتون از تابع range برای iterate کردن استفاده میکنیم. فعلا نیاز نیست معنی این اصطلاحات را بدانید. اینها را جلوتر یاد میگیرید. پس تعداد تکرار for loop ها به مقدار عددی که در ()range گذاشته شود بستگی دارد. نکته جالبی که باید بدانید این اسن که i در گردش اول 0 است. در گردش دوم 1 و به ترتیب تا هر مقدار باری که تکرار میشود. و شما میتوانید از همین استفاده کنید تا برای مثال اعداد 1 تا 10 را نمایش دهید.

سوال : اگر به شما یک لیست بدهم و از شما بخواهم که با یک حلقه تک تک اعضای آن را پرینت کنید چه میکنید ؟ (راهنمایی : برای بدست آوردن طول یک لیست از تابع ()len استفاده میکنیم.)

l = [1, 2, 3, 6, 9, 3, 4, 7, 1, 3, "a", "c", None, True]

اگر کدتان چیزی مشابه کد زیر است کارتان را درست انجام داده اید.

l = [1, 2, 3, 6, 9, 3, 4, 7, 1, 3, "a", "c", None, True]
for i in range(len(l)):
    print(l[i])

البته این کد اشکالی بزرگ داره و اونم اینه که این کد پایتونیک نیست. این کد در زبان ها دیگه کاملا درست است . در پایتون هم درست اس ت اما پایتونیک نیست. یعنی راه بهتری و ساده ترسبرای انجام این کار وجود دارد. تابع ()range یک iterator (که در آینده یاد میگیرید و فعلا میتوانید آن را یک لیست فرض کنید) بر میگرداند.

اینجا بهترین کاری که متوانید بکنید کد زیر است : 

l = [1, 2, 3, 6, 9, 3, 4, 7, 1, 3, "a", "c", None, True]
for i in l:
    print(i)

در کد بالا i برابر با همان آیتم های لیست است. یعنی در دوره اول آیتم اول، در دوره دوم آیتم دوم و ...

از همین میتوانیم نتیجه بگیریم که تابع ()range در واقع یک لیست ازاعداد 0 تا n بر میگرداند.

حلقه While :

کلمه while در انگلیسی یعنی تا وقتی که. در برنامه نویسی نیز به همین معنی است. مثال :

while a < 10:
    print("a")
    a += 1

تبدیل انواع داده به یکدیگر

در خیلی از اوقات نیازداریم که انواع داده را به یکدیگر تبدیل کنیم. برای تبدیل داده ها به یکدیگر تابع هایی از قبل تعریف شده مانند ()int و ()str و ()list و ()dict و ... مثال : 

a = 10
s = str(10)
print(s)
c = 1
b = bool(c)
print(b)
d = "123"
i = int(123)
print(i)
e = "123a"
f = int(e)
print(f)

و خروجی : 

10
True
123 
Traceback (most recent call last):
  File "c:\Address\To\Your\File\main.py", line 11, in 
    f = int(e)
ValueError: invalid literal for int() with base 10: '123a'

در مثال بالا یک ارور مشاهده میکنید و آن هم به خاطر این است که داده "" یک حرف a دارد که نمیتواند تبدیل به عدد شود و به خاطر همین ارور می دهد.

آمادگی برای ارور ها :

خیلی از اوقات ما در کد به ارور بر میخوریم و دلیل آن هم از کد خود ما نیست.چون پیشبینی میکنیم که این ارور اتفاق خواهد افتاد پس از وقوع آن جلوگیری میکنیم. در پایتون از try و except استفاده میکنیم .مثال : 

a = "123a"
try:
    b = int(a)
    print(b)
except Exception as e:
    print(f"Exception Happenned :\n{e}")

خروجی : 

Exception Happenned :
invalid literal for int() with base 10: '123a'

در کد بالا چون "123a" دارای حرف a است نمیتواند تبدیل به عدد شود و به همین خاطر ارور رخ میدهد. حال میتوانیم با try کدرا امتحان کنیم و اگر اروری رخ داد نوع ارور را در e ذخیره میکنیم و سپس آن را پرینت میکنیم.

نکته : میتوانید نوع ارور را مشخص کنید برای مثال مثال بالا را به شکل زیر نیز میتوان نوشت : 

a = "123a"
try:
    b = int(a)
    print(b)

    print(f"Exception Happenned :\n{e}")
